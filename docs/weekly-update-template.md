# Weekly Update Template

Post every Friday to GitHub Discussions (Announcements category).

## Title Format

```
Kai Weekly Update — Week N
```

## Body Template

```markdown
## What shipped

- [Feature/fix description with context]
- [Feature/fix description with context]

## What we're validating

- [Thing being tested or measured]
- [Thing being tested or measured]

## Metrics

- [Key number]: [value]
- [Key number]: [value]

## Bugs found / fixed

- [Bug description and resolution]

## Lessons learned

- [Insight from the week's work]

## Next week

- [Priority 1]
- [Priority 2]
- [Priority 3]
```

## Guidelines

- No fluff. No hype. Ship notes and numbers.
- Link to relevant issues and PRs.
- Include benchmarks or metrics when available.
- Keep it under 500 words.
- Optional: architecture diagram, demo clip, debugging story.

## Where to post

1. GitHub Discussions → Announcements
2. Slack → #announcements
3. Twitter/X
4. LinkedIn
